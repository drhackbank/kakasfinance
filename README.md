## GitHub Pages pe Deploy — 3 Steps

### Step 1 — GitHub pe repo banao

1. **github.com** pe jaao → login karo
2. Top-right **"+"** → **"New repository"**
3. Repository name: **`kakasfinance`**
4. Visibility: **Public** (GitHub Pages free mein sirf public pe kaam karta hai)
5. **"Create repository"** click karo

---

### Step 2 — Files upload karo

**Option A — Browser se (easiest):**
1. Naye repo mein **"uploading an existing file"** link click karo
2. Sab files drag & drop karo (index.html, style.css, script.js, package.json, public/ folder)
3. **"Commit changes"** click karo

**Option B — Terminal se:**
```bash
cd kakasfinance-folder
git init
git add .
git commit -m "kakasfinance init"
git remote add origin https://github.com/YOUR_USERNAME/kakasfinance.git
git branch -M main
git push -u origin main
```

---

### Step 3 — GitHub Pages enable karo

1. Repo mein **Settings** tab click karo
2. Left sidebar mein **Pages** click karo
3. **Source** → **"Deploy from a branch"** select karo
4. **Branch** → **main** select karo, folder **/ (root)** rakho
5. **Save** click karo
6. 2-3 minutes wait karo

**Tumhara site live hoga:**
```
https://YOUR_USERNAME.github.io/kakasfinance/
```
